Responses to Questions:

  • Name another method (not search engines) that is used to navigate the WWW.
  • Hyperlinks.
  • Explain the purpose of BOTS/SPIDERS.
  • BOTS, sometimes known as spiders, are automated pieces of code that crawl over the web and website, cataloging the pages they find for page ranking (web browser search results) or data collection.
  • How can you as a webmaster prevent your site from being crawled.
  • Adding a mandatory login page to access the website which prevents bots from getting in if they don't know the password, or using a robots.txt to ask the crawlers to kindly stop.
  • List 3 ways that you can improve the ranking of a web site.
    • Only include relevant and correct information on the site.
    • Making a nicely layed-out page that increases user retention.
    • Frequently updating the page to show recent activity.
  • List 3 ways that you can cause a web site to lose ranking.
    • Creating a giant network of pages that all link to each other to increase inlink/outlink counts, which when find out will get you IP banned.
    • Making a crappy website that is gross to navigate and drops user retention rates.
    • Onky spamming relevant keywords in the site description and body.
